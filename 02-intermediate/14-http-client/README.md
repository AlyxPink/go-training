# Exercise 14: HTTP Client

**Difficulty**: ⭐⭐ Intermediate
**Estimated Time**: 55 minutes

## Learning Objectives

- Master net/http client package
- Handle HTTP requests and responses
- Configure timeouts and headers
- Parse JSON responses
- Handle errors properly

## Problem Description

Build HTTP client utilities for making requests to REST APIs.

### Requirements

1. **GET request** - Fetch data from API
2. **POST request** - Send JSON data
3. **Custom headers** - Add authentication, user-agent
4. **Timeouts** - Configure request timeouts
5. **Error handling** - Handle network errors, status codes
