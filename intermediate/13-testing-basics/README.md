# Exercise 13: Testing Basics

**Difficulty**: ⭐⭐ Intermediate
**Estimated Time**: 60 minutes

## Learning Objectives

- Master testing package
- Write table-driven tests
- Use subtests with t.Run
- Understand test coverage
- Practice benchmark writing

## Problem Description

Learn Go testing patterns by writing comprehensive tests.

### Requirements

1. **Table-driven tests** - Multiple test cases in one test
2. **Subtests** - Organize tests with t.Run
3. **Helper functions** - Mark test helpers with t.Helper()
4. **Benchmarks** - Performance testing
5. **Test coverage** - Measure code coverage

## Testing
```bash
go test -v
go test -cover
go test -bench=.
```
