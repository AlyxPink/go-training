# Exercise 05: File Operations

**Difficulty**: ⭐⭐ Intermediate
**Estimated Time**: 50 minutes

## Learning Objectives

- Master os package for file operations
- Use bufio for efficient I/O
- Handle file metadata and errors
- Practice safe file handling patterns

## Requirements

1. **FileInfo** - get file size, modification time, permissions
2. **ReadLines** - read file line by line efficiently
3. **WriteLines** - write slice of strings to file
4. **CopyFile** - copy file with error handling
5. **WalkDirectory** - list files recursively
